# IT Helpdesk (ระบบแจ้งซ่อมไอที)

โปรเจกต์ Web Application สำหรับจัดการระบบแจ้งซ่อมและรับเรื่องร้องเรียนปัญหาไอทีภายในองค์กร ช่วยให้พนักงานแจ้งปัญหาได้ง่ายขึ้น และให้ทีมช่าง (IT Support) ติดตามงานและอัปเดตสถานะได้อย่างเป็นระบบ

## Tech Stack
* **หน้าบ้าน (Frontend):** React.js, Tailwind CSS (อัปโหลดขึ้น Vercel)

## วิธีติดตั้ง
1. ติดตั้งแพ็กเกจ: npm install
2. รันเซิร์ฟเวอร์จำลอง: npm run dev
